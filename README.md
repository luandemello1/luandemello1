## 💡 Exemplos de Código

### Lua
```lua
-- Função para calcular o fatorial de um número
function fatorial(n)
  if n == 0 then
    return 1
  else
    return n * fatorial(n - 1)
  end
end

print('Fatorial de 5:', fatorial(5))
```

### JavaScript
```javascript
// Função para calcular o fatorial de um número
function fatorial(n) {
  if (n === 0) {
    return 1;
  } else {
    return n * fatorial(n - 1);
  }
}

console.log('Fatorial de 5:', fatorial(5));
```
